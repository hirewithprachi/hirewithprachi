Step 4: Import GTM Configuration

1. Go to your Google Tag Manager account

2. Click "Admin" ‚Üí "Import Container"

3. Upload the gtm-config.json file from this project

4. Choose "Merge" option

5. Review and publish

Step 5: Update Analytics Configuration

Update the GTM ID in src/lib/analytics.js:

this.gtmId = 'GTM-XXXXXXX'; // Replace with your actual GTM ID

üß™ Testing

Test GTM Installation

1. Open your website

2. Open browser developer tools (F12)

3. Go to Console tab

4. Type: window.dataLayer

5. You should see an array (even if empty)

6. If you see "undefined", GTM is not installed correctly

Test GTM Preview Mode

1. In GTM, click "Preview"

2. Enter your website URL

3. Click "Start"

4. Visit your website in the new tab

5. Check GTM preview panel for events

üìä Verification

Check Google Analytics

1. Go to Google Analytics

2. Navigate to "Reports" ‚Üí "Realtime"

3. Visit your website

4. You should see real-time data

‚úÖ Success Indicators

  • GTM loads without errors in browser console
  • Data layer is accessible via window.dataLayer
  • Real-time data appears in Google Analytics
  • Events are firing in GTM preview mode

üîß Troubleshooting

Common Issues

  • GTM not loading: Check if the GTM ID is correct
  • No data in GA: Verify GA4 property and data stream settings
  • Events not firing: Check GTM triggers and tags configuration
  • Console errors: Ensure GTM code is placed correctly
